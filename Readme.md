## Teleports to last boss and auto reset dungeon only ace ones.

---

## Supported Dungeons:
* Ghilieglade
* Ace-Akasha
* Ace-Baracos
* Channelworks
* Sky Cruiser Extreme

---

## Command
* Type /8 solo to activate or deactivate the module.
* Module is enabled by default.

---

Auto reset after looting but only in the ace ones.
